## Important .SE Domain Background

### WHOIS Privacy — Already Built Into .SE
The Swedish Internet Foundation (Internetstiftelsen) **automatically hides personal data** in WHOIS lookups for all .SE domains, in compliance with GDPR. The registry's own privacy policy confirms that personal information is not publicly disclosed. **You do NOT need to purchase WHOIS privacy for .SE domains** — it is effectively included at the registry level. Some registrars may still list "WHOIS protection" as a feature, but it provides no additional benefit for .SE.

### DNSSEC for .SE
Internetstiftelsen supports **automated DNSSEC provisioning via CDS records**, allowing key management without manual registrar interaction. Many Swedish registrars offer automatic DNSSEC signing at no extra cost. DNSSEC is highly recommended for .SE domains.

### Registry Lock for .SE
Registry Lock is an **optional paid service** offered by Internetstiftelsen. It prevents unauthorized changes to domain registration data. It is **not mandatory** for registrars to offer it, and availability varies. Price is set by each registrar.

### Snapback/Backorder for .SE
.SE domains that expire go through a deletion process. Specialized services (e.g., Web Lovers) offer drop-catching/snapback for .SE domains. Not all registrars offer this.

## Key Takeaways

1. **For the absolute cheapest .SE domain:** HostUp at €9.90/yr first year, €13.50/yr renewal — AND includes free Registry Lock.
2. **For best features with domain-only:** Simply.com offers the most complete package (API, DNSSEC, email forwarding, URL forwarding, backorder) at a reasonable $19.90/yr.
3. **WHOIS Privacy is a non-issue** — .SE already hides personal data at the registry level. Don't pay extra for this.
4. **DNSSEC should be enabled** — Most Swedish registrars offer it free. Make sure your chosen registrar supports it.
5. **Registry Lock is rare** — HostUp is the only registrar found offering free Registry Lock for .SE. If security is paramount, this is significant.
6. **Avoid GoDaddy for .SE** — No inbound transfers, no backorders, and premium pricing.
7. **Cloudflare and Hetzner don't support .SE** — Use them for DNS/CDN instead.
